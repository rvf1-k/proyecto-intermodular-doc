# Introducción

## Objetivo del proyecto

El objetivo del proyecto es desarrollar Forum The Forums (FTF), traducido a español como Foro de foros, es un foro/red social en el que los usuarios pueden comentar directamente sobre el contenido de páginas web a través de una extensión.

## Finalidad

Crear una forma de tener discusiones superpuestas a Internet, funcionando de manera similar a Reddit o Twitter, pero sobre páginas web específicas. Los usuarios podrán seleccionar texto o imágenes, comentar sobre ellas y generar hilos de discusión. Además, de poder responder, compartir y debatir en la página principal de FTF.

Estos comentarios no solo aparecerán en la plataforma de FTF, sino que también serán visibles para otros usuarios de FTF que visiten la misma web, como comentarios flotantes o un embed.

## Legalidad

Muchas páginas webs que en su TOS ponen restricciones sobre el uso de su contenido, pero FTF no altera directamente el contenido, solo añade una capa de información extra, sobre IP y Copyright citar fragmentos cortos de contenido se considera uso legítimo como es el "[fair use](https://es.wikipedia.org/wiki/Uso_justo)" en EE.UU. o "[cita legal](https://es.wikipedia.org/wiki/Derecho_de_cita)" en la UE, por lo que se limita la cantidad de texto.

# Problemas

Debido a las limitaciones, no se tiene como propósito el producto final si no como MVP (mínimo producto viable), cumpliendo con el diseño y las principales funciones.
