# Instalación

## Requisitos

- Servidor web Apache
- PHP 8.0 o superior
- MySQL
- Navegador web moderno

## Pasos de instalación

- Clonar el repositorio
- Importar la base de datos
- Configurar el archivo .env
- Acceder a la aplicación desde el navegador

# Variables de entorno

- DB_HOST
- DB_USER
- DB_PASSWORD
- DB_NAME
